# 五号员工大级别结构形态识别规格

本文件是五号员工30+/60+维度归因的上游规格。五号员工不是只罗列指标，而是必须先识别大级别结构形态，再把结构拆成可统计维度。

用户给出的“月线/周线头肩顶后长周期下跌磨底，再在凹口附近巨震、大阳攻击、盘整数月、最终突破左侧前高”的案例，只是一个结构范例，不是唯一模型。五号员工必须建立结构形态库，持续从大涨样本中总结可复用共性。

## 1. 最高原则

五号员工深度归因必须先回答：

```text
它属于哪一类大级别结构？
这个结构在涨之前留下了哪些端倪？
这些端倪能不能被量化？
这些端倪能不能在多只大涨样本中统计共振？
这些端倪能不能转化为一号员工后续选股优化因子？
```

如果看不出结构，不能强行归因；必须标记为“未识别明确结构形态”，并说明缺少哪些证据。

## 2. 结构识别优先级

五号员工归因顺序：

1. 先看季线/月线/周线大级别结构。
2. 再看日线是否出现触发K、突破、回踩、承接。
3. 先识别形态原型，再拆成维度。
4. 先找结构核心线，再看量能和K线质量。
5. 先判断是否真实可量化，再决定是否计入维度。

不能反过来先堆指标，再硬凑结构。

## 3. 结构原型字段模板

每一种结构原型都必须输出以下字段：

```text
archetype_id：结构编号
archetype_name：结构名称
cycle_level：日/周/月/季/年
formation_bars：结构形成K线数量
left_structure：左侧结构
breakdown_or_retest：跌破/回抽/试盘过程
bottom_or_absorption：底部磨底/吸收过程
pre_launch_clues：启动前端倪
trigger_event：触发事件
volume_signature：量能签名
core_level：核心结构线
invalid_evidence：反证/不成立条件
convertible_factors：可转化为一号员工的因子
```

## 4. 大级别结构形态库

### AR01 大级别头肩顶供给释放后再启动

这是用户举例的核心结构之一，但不是唯一结构。

识别要点：

- 周线/月线/季线形成清晰头肩顶或类头肩顶。
- 形成时间可长达二三十个月、三四十个月，甚至更长。
- 左肩、头部、右肩结构清晰，右肩不明显弱于左肩时称为强势头肩顶。
- 头肩顶区域量能饱和，说明当时发生过充分筹码交换。
- 后续跌破颈线或平台下沿，进入长周期下跌或底部磨底。
- 下跌后若成交量长期萎缩，说明旧供应释放后进入低关注/低换手阶段。
- 启动前若月线/周线连续数根K线开始抬高、量能恢复，属于重要端倪。
- 若在原头肩顶颈线、凹口、右肩、左峰附近出现大阳线、巨震、放量攻击，说明资金开始重新挑战历史供应区。
- 大阳攻击后若再盘整数月不深跌，属于供应吸收确认。
- 最终若突破左侧前高、头肩顶上沿或关键凹口，属于大级别历史供应区被打穿。

可量化字段：

- 头肩顶形成月数。
- 左肩/头部/右肩高点价差百分比。
- 颈线斜率和跌破幅度。
- 顶部区域量能分位/成交量饱和度。
- 跌破后下跌持续月数和最大跌幅。
- 底部磨底持续月数。
- 底部量能相对顶部量能萎缩比例。
- 启动前3到6个月价格重心斜率。
- 启动前3到6个月量能恢复比例。
- 凹口附近大阳线数量、实体质量和量能强度。
- 凹口攻击后盘整月数和回撤深度。
- 最终突破左侧前高/颈线/头部高点的实体质量。

可转化因子：大级别头肩顶历史供应吸收因子、颈线回收因子、凹口攻击因子、长周期磨底后量能恢复因子。

### AR02 大级别平台跌破后破底翻

识别要点：

- 月线/周线存在清晰平台上下沿。
- 曾经有效跌破平台下沿，市场认为结构失败。
- 后续长期底部吸收后重新站回平台下沿或跌破大阴K实顶。
- 重新站回时量能恢复，K线实体质量好。
- 站回后不快速跌回平台下方。

可量化字段：平台持续月数、上下沿实体共振次数、跌破幅度、跌破K量能、底部吸收周期、重新站回实体比例、回踩不破天/月数。

### AR03 历史最大量阳K高点/实底再突破

识别要点：

- 周线/月线/季线历史最大量K必须为有效阳K。
- 该K高点形成历史供应线，实体实底形成资金防守线。
- 后续多年围绕该线反复试探、失败、吸收。
- 真正启动前，日线/周线高质量突破该最大量阳K高点。

可量化字段：最大量K排名、实体占比、影线比例、高点/实底反应次数、突破实体比例、突破量能、突破后是否回落。

### AR04 远期最大量高点 + 更高试盘高点 + 长时间消化 + 二次确认

识别要点：

- 历史最大量阳K形成绿线。
- 后续某根关键K创出更高试盘高点，但月线/周线收盘未真正站稳。
- 这次试盘不是失败废点，而是未来确认线。
- 经过一倍、二倍或更长时间消化后，日线/周线高质量突破试盘高点。

可量化字段：最大量K到试盘K间隔、试盘K到再启动间隔、时间倍数、试盘K量能强弱、试盘失败后的回撤、再突破实体质量。

### AR05 大级别圆弧底/碗形底后颈线突破

识别要点：

- 月线/周线长期形成圆弧底或碗形底。
- 左侧下跌后波动逐渐收敛，底部成交量萎缩。
- 右侧价格重心逐渐抬升，量能温和恢复。
- 颈线或碗沿突破时出现健康放量和强实体。

可量化字段：底部持续月数、低点曲率、右侧斜率、量能恢复斜率、颈线反应次数、突破实体质量。

### AR06 双底/W底后右侧突破

识别要点：

- 大级别形成两个或多个低点，第二低点不再创新低或假破后收回。
- 第二低点成交量更低或下影更强，说明卖压衰竭。
- 右侧突破中轴线/颈线后进入加速。

可量化字段：两底间隔、价差、第二底量能、下影反应、颈线突破实体比例、突破后承接。

### AR07 长周期箱体吸收后一次性打穿上沿

识别要点：

- 月线/周线长期横盘箱体。
- 箱体内多次上沿受压、下沿承接。
- 箱体后段量能平稳、波动压缩。
- 突破日/月K实体大部分位于箱体上沿之上。

可量化字段：箱体持续时间、上下沿触碰次数、箱体宽度、量能CV、突破实体占比、突破量比。

### AR08 台阶式平台推进

识别要点：

- 每上一个台阶后形成整理平台。
- 右侧平台价格中枢高于左侧平台。
- 右侧平台均量也高于左侧平台，说明资金成本抬升。
- 整理时不跌破上一启动K实底或中位。

可量化字段：平台数量、平台中枢抬升幅度、平台均量抬升幅度、关键启动柱承接、放量长阴数量。

### AR09 核心压力线多次试盘后真突破

识别要点：

- 同一核心压力线附近多次上影、冲高回落或放量滞涨。
- 前几次试盘失败并不一定坏，可能是在扫流动性和消化供应。
- 真突破必须实体站稳、收盘强、上影短，并最好突破前次失败高点。

可量化字段：试盘次数、失败高点、失败量能、上影比例、再次突破实体质量、突破是否打穿失败高点。

### AR10 凹口攻击 + 再平台 + 最终突破

识别要点：

- 大级别左侧有明显凹口、颈线或断层压力区。
- 右侧第一次靠近凹口时出现巨震、大阳线或放量攻击，显示资金意图。
- 攻击后不直接突破，而是盘整数周/月吸收供应。
- 最终再次突破凹口或左峰时进入主升。

可量化字段：凹口高度、第一次攻击实体、第一次攻击量能、攻击后盘整周期、盘整回撤、二次突破质量。

### AR11 BOLL/BBI大周期缩口修复后爆发

识别要点：

- 月线/周线BOLL缩口处于低分位。
- 缩口后价格围绕中轨反复支撑或跌破再修复。
- 重新站回中轨后，日线触发突破。

可量化字段：BOLL带宽分位、中轨维持月数、跌破后修复月数、日线触发质量。

### AR12 长期下跌后量能枯竭 + 量能恢复

识别要点：

- 经历长期下跌或漫长磨底。
- 下跌后成交量明显萎缩，极端活跃资金退出。
- 启动前几个月量能逐渐恢复，价格重心抬高。
- 恢复期不出现破坏性放量长阴。

可量化字段：下跌持续时间、底部量能/顶部量能比、近3/6个月量能恢复斜率、价格重心斜率、放量长阴数量。

### AR13 强势头肩顶失败后的再突破

识别要点：

- 头肩顶当时并非纯弱势，而是高位强势分歧结构：量能饱和、右肩不弱、反复争夺。
- 后续虽跌破，但底部回收后再突破右肩/头部附近，说明历史强供应被重新吸收。
- 这种结构若配合大级别量能恢复和凹口攻击，启动弹性更强。

可量化字段：右肩相对左肩高度、右肩量能、跌破后回撤深度、回收右肩线质量、头部高点突破质量。

### AR14 多周期结构共振爆发

识别要点：

- 月线识别大级别历史供应区。
- 周线识别平台/凹口/核心线。
- 日线识别触发K、突破、承接。
- 三个周期信号方向一致。

可量化字段：月线结构分、周线结构分、日线触发分、周期冲突标记、共振得分。

## 5. 形态归因输出要求

五号员工每只深度样本必须输出：

```text
主结构原型：ARxx
辅助结构原型：ARxx/ARxx
大级别周期：月线/周线/季线
核心结构线：价格、来源、反应次数
启动前端倪：价格、量能、K线、波动、时间
触发确认：突破哪条线、实体质量、量能质量
反证：为什么不是这个结构，或者结构证据不足
```

若无法识别任何AR结构，必须写：

```text
主结构原型：未识别
原因：缺少大级别结构、缺少核心线、缺少量能证据、历史数据不足等
```

不能为了凑30维强行把股票套进某个形态。

## 6. 与30+/60+维度的关系

结构原型不是替代维度，而是维度的上游。

正确顺序：

```text
先识别AR结构原型
再拆成时间、结构、量能、压力线、触发K、风险反证等维度
最后统计3只样本共振维度
```

如果某个AR结构在多日大涨样本中反复出现，就应该进入一号员工重点优化方向。

## 7. 对一号员工的转化方向

五号员工长期统计后，应向一号员工输出：

- 高频AR结构排行榜。
- 每个AR结构的胜率/涨幅/回撤统计候选。
- 启动前最常出现的端倪组合。
- 最容易误判的反证条件。
- 值得写入一号员工基础召回和深度评分的因子。

五号员工不是推荐买入，而是为一号员工总结大涨样本背后的结构规律。
