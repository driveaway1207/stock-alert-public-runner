## 0. 五号员工最新身份

五号员工不是选股打分器，不是荐股员工，不是交易员工。

五号员工是：

```text
大涨原因口径池研究员
涨停/大涨样本归因员
民间战法与华尔街体系映射员
暴涨逻辑维度池沉淀员
```

五号每天研究涨停、大涨、强势样本，不给股票打买入分，而是识别每只大涨样本命中了哪些上涨原因口径，并长期统计这些原因的出现频率、共振关系、组合关系、反证条件和事前可观察性。

## 2. 五号不是股票评分表

五号不输出：

```text
股票综合分
买入分
推荐分
交易优先级分
明天可关注分
```

五号输出：

```text
原因口径命中
原因口径共振统计
原因组合统计
未命中/反证提示
事前可观察性
是否只是事后归因
```

“维度”必须是能解释暴涨前因果链的原因口径，不是涨幅字段、普通突破字段、调试字段、同源重复字段。

## 3. 民间战法与华尔街体系融合原则

五号必须融合两套体系：

```text
用户的民间实战战法
+
华尔街专业市场结构框架
=
统一的底层市场行为原因口径
```

不能只听民间战法，也不能只堆华尔街术语。二者必须映射到底层市场行为。

如果民间战法和华尔街表达描述的是同一个市场行为，只能归并成一个原因口径，禁止重复计分、重复论证。

示例：

```text
民间：物极必反
华尔街：high-volume impulse failure / failed acceptance / no follow-through demand
底层行为：最高级别主动攻击没有被市场接受，后续形成失败事件
统一口径：R001 大周期高量强阳失败/物极必反
```

## 4. 周期映射硬规则

用户人工复盘中说的“月线”，在五号聚合K体系中优先映射为：

```text
20日聚合K ≈ 月线级别
60日聚合K ≈ 季线级别
250日聚合K ≈ 年线级别
```

所有“月线倍量、月线平量、月线实体质量、月线双上影共振、月线量价合一”等描述，在五号代码里优先按 20日聚合K 计算。

如果未来额外使用自然月K，必须单独标注，不得和 20日聚合K 混算。

## 5. 暴涨原因口径池第一批：R001-R014

本批口径来自用户对 688260 的人工复盘。688260 不是孤立案例故事，而是用于拆解可复用的暴涨原因维度池。后续五号必须继续扩充，不得把本批口径当最终全集。

每个口径都必须包含：民间表达、华尔街映射、底层市场行为、量化条件、反证条件、事前可观察性。

### R001 大周期高量强阳失败/物极必反

民间表达：最大量大阳线本来级别很高，后面却被缩量阴线组合跌坏，甚至跌破虚底，属于物极必反。

华尔街映射：high-volume impulse failure / failed acceptance / no follow-through demand。

底层行为：最高级别主动攻击没有被市场接受，前一次强事件失败。

量化条件：

```text
1. 20日聚合K/60日聚合K出现阶段最大量或局部极高量强阳。
2. 实体涨幅大、收盘位置强、上影不长。
3. 后续若干聚合K跌破该强阳实体底或最低点。
4. 跌破过程若是缩量或量能逐步衰减，更符合“强攻击失败后消化”逻辑。
```

反证：强阳后继续站稳并抬升，不算失败；若后续极端放量长阴砸穿，要另归类为放量破坏。

### R002 强阳失败后的二次努力供应线

民间表达：第一次最大努力失败后，后面再次放出阶段最大量努力上攻，但仍然只打到一个更低高点，这个高点很重要。

华尔街映射：effort vs result / lower high on renewed volume / supply reaction level。

底层行为：资金第二次努力仍未解放左侧套牢区，形成供应极限。

量化条件：

```text
1. 前面已经有 R001 高量强阳失败。
2. 随后进入缩量消化。
3. 缩量后出现阶段最大量或局部显眼量K。
4. 该K向上测试但未突破前高或未解放前次高量强阳筹码区。
5. 取其最高点作为二次努力供应线候选。
```

### R003 隔断式长周期带量双上影线共振核心线

民间表达：两个相隔很久的带量上影线高点在同一价格区间共振，中间可以有阶段性隔断，但不能多年乱穿。

华尔街映射：market memory level / supply reaction confluence / clustered breach tolerance。

底层行为：多个历史阶段在同一区域发生供应反应，形成长期市场记忆。

量化条件：

```text
1. 左侧存在带量上影线高点。
2. 右侧存在二次努力高点或另一根带量上影高点。
3. 两者价格差落在同一容忍带内。
4. 两端对应量能都必须在当时局部阶段显眼。
5. 中间允许集中穿越，但穿越必须聚集在某一小段时间。
6. 不能多年、多阶段、随机反复穿越。
```

反证：中间每隔几年就反复穿越，降级；两端量能不起眼，降级；价格差太大，不能强凑共振。

### R004 局部相对量能确认原则

民间表达：历史量能不能跨年份绝对比较，2022年的量不能拿去和2026年的量直接比；要看它在当时是不是显眼。

华尔街映射：relative volume / local volume regime / contextual volume significance。

底层行为：历史反应点的量能价值取决于当时环境，不取决于跨年份绝对大小。

量化条件：

```text
1. 当前K量能 / 前1根量能。
2. 当前K量能 / 前5、10、20根均量。
3. 当前K量能在近N根中的排名。
4. 当前K量能相对后续几根是否明显突出。
5. 若20日聚合K样本不足，则下钻周线/日线验证。
```

硬规则：不能因为历史量能小于当前量能，就否定左侧共振点的带量级别。

### R005 大周期并肩高倍量平量强阳结构

民间表达：月线/周线/季线级别两个相邻高倍量平量，尤其第一根实体好，级别很高。

华尔街映射：volume consistency + sustained initiative demand + price acceptance。

底层行为：连续两个高周期里资金参与强度高度一致，且价格推进有效。

硬定义：

```text
1. 两根K线必须严格相邻，中间不能有任何穿插；不相邻不叫并肩。
2. 两根都属于局部高倍量或显眼高量。
3. 两根量差：
   强平量 ≤3%-5%
   合格平量 >5% 且 ≤8%
   宽松平量 >8% 且 ≤15%
   超过15% 不叫平量，只能叫同级别高量或相近量。
4. 至少第一根是高质量强阳实体。
5. 第二根不能破坏第一根强阳结构。
```

第二根“不破坏”的融合定义：

```text
S级：第二根收盘在第一根实体上三分之一位之上。
A级：盘中刺破上三分之一，但收盘站回。
B级：收盘跌破上三分之一，但仍在实体中位之上。
C级：跌破实体中位，明显降级。
D级：跌破第一根实底，强阳失败。
```

### R006 大周期高倍量量价合一

民间表达：月线/20日聚合K大倍量必须配得上大实体、短上影和强收盘，否则就是有量无价。

华尔街映射：effort vs result。

底层行为：量能代表努力，实体推进、收盘位置、上影线代表结果；高量低效要归为风险。

量化字段：

```text
vol_ratio：当前聚合K量 / 局部均量
entity_pct：实体涨幅
body_ratio：实体 / 全振幅
close_pos：收盘位置
upper_shadow_ratio：上影线 / 全振幅
follow_through：后续承接
```

初始判断口径：

```text
20日聚合K 2-3倍量：实体涨幅最好 ≥15%-20%。
20日聚合K 3-5倍量：实体涨幅最好 ≥20%-30%。
20日聚合K 5倍以上：实体涨幅最好 ≥30%，收盘位置要强，上影线不能长。
```

反证：高倍量 + 小实体 = 高量低效；高倍量 + 长上影 = 供应压制；高倍量 + 后续跌破实底 = 强攻击失败。

### R007 逻辑分析线

民间表达：白线不是核心线，但它解释资金状态如何从冷转热，解释右侧量价演化和大涨前因果链。

华尔街映射：change-of-character anchor / regime shift reference。

底层行为：某条线不是最终核心压力，但能解释后续量价演化、启动路径和资金状态切换。

688260层级：

```text
绿线 = 核心线。
白线 = 逻辑分析线。
```

白线可来自：最大强阳实体实底 + 并肩高倍量平量强阳结构 + 右侧量能改善 + 右侧多点共振。

硬规则：白线不能写成核心线；它是解释右侧资金逻辑的分析线。

### R008 启动式高倍量平量后的右侧量能改善

民间表达：白线成立前量能很差，成立后右侧量能逐渐好转。

华尔街映射：volume regime shift / change of character。

底层行为：资金参与状态从低迷转为活跃。

量化条件：

```text
1. 左侧量能低迷。
2. 出现高倍量并肩平量强阳结构。
3. 右侧若干聚合K均量高于左侧均量。
4. 右侧攻击、回踩、试盘开始有更稳定量能参与。
```

### R009 逻辑线跌破后的规律性缩量小阴

民间表达：白线后面被三根小阴跌破，但三根小阴量能平稳萎缩，缩量比例有规律。

华尔街映射：controlled pullback / orderly volume decay。

底层行为：价格回落，但没有恐慌抛压，资金状态仍可控。

量化条件：

```text
1. 逻辑线被小阴线组合跌破。
2. 下跌K线实体不大。
3. 量能逐级缩小。
4. 缩量比例相近，例如连续每根缩10%-15%附近。
5. 右侧下跌量能均值仍高于左侧低迷期均量。
```

反证：放量长阴跌破 = 破坏；缩量无规律且价格快速崩坏 = 降级。

### R010 逻辑分析线右侧多点共振

民间表达：白线画出后，右侧锤子线、实体、上影线、阶段高量K都和白线有关系。

华尔街映射：multi-touch validation / reaction clustering。

底层行为：这条分析线被后续市场反复记住。

量化条件：

```text
1. 白线生成后，右侧多个K线实体、影线或收盘价接近白线。
2. 至少有一个反应点带量。
3. 这些反应不是随机散乱，而与后续资金结构变化有关。
```

### R011 规则阳梯量推进

民间表达：突破白线前后，阳量逐级递增，递增比例有规律，最后一根量最大、实体也最大。

华尔街映射：progressive demand expansion / initiative buying buildup。

底层行为：资金以有节奏的方式逐步增强，价格推进同步增强。

量化条件：

```text
1. 连续多根阳量递增。
2. 每根递增幅度相对平滑，不是忽大忽小。
3. 最后一根阳量最大。
4. 最后一根对应K线实体也最大或最强。
5. 上影线不能太长，收盘位置要强。
```

反证：量递增但实体越来越小 = 高量低效；量递增但长上影 = 供应压制；递增关系杂乱 = 降级。

### R012 核心线突破后的接受确认

民间表达：突破绿线后，后面多根量能还在均量之上，没有快速退潮。

华尔街映射：acceptance above breakout level / post-breakout participation。

底层行为：突破后市场没有快速退潮，而是在新价格区间继续成交。

量化条件：

```text
1. 有效突破核心线。
2. 突破后若干根K线成交量保持在局部均量上方。
3. 价格没有快速跌回核心线下方。
```

### R013 核心线回踩支撑确认

民间表达：后面有大阴量、大阴实体，但最后在绿线上获得支撑。

华尔街映射：breakout-retest / resistance turns support。

底层行为：原核心压力线被突破后，回踩时转成支撑，说明供应被吸收。

量化条件：

```text
1. 先突破核心线。
2. 后续回落接近核心线。
3. 收盘不有效跌破核心线。
4. 最好后续出现修复K线。
```

### R014 核心线上方平量阳包阴

民间表达：绿线支撑后出现平量阳包阴，量能非常平稳，完成稳定反包。

华尔街映射：stable participation reversal / reversal confirmation。

底层行为：核心线支撑后，资金用稳定量能完成价格修复。

量化条件：

```text
1. 前面有核心线支撑确认。
2. 出现阳线反包前一根阴线。
3. 反包量能与前一根或附近关键量能接近平量。
4. 平量阈值仍按严格标准：强平量≤3%-5%，合格≤8%，宽松≤15%。
5. 阳线实体质量要好。
```

## 7. 数据与样本原则

涨停池必须尽量全量，覆盖主板、中小板、创业板、科创板、北交所、ST 和非 ST。

涨停阈值原则：ST约5%，主板/中小板约10%，创业板/科创板约20%，北交所约30%。报告必须输出涨停总数、板块分布、10cm/20cm/30cm分布、北交所数量、数据来源和失败数量。

普通涨停只做统计，不堆股票名单；Telegram主报告只写大类统计和原因口径归因，完整名单放 artifact。

深度样本默认取：对可取得历史K线的涨停样本，按20日/月线窗口涨幅排序，取前三只做深度归因。若样本为空，必须说明原因，不能伪造。
