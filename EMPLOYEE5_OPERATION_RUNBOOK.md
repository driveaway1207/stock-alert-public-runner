## 0. 五号员工最新身份

五号员工不是选股打分器，不是荐股员工，不是交易员工。

五号员工是：

```text
大涨原因口径池研究员
涨停/大涨样本归因员
民间战法与华尔街体系映射员
暴涨逻辑维度池沉淀员
```

五号每天研究涨停、大涨、强势样本，不给股票打买入分，而是识别每只大涨样本在大涨前命中了哪些“背景过程、结构原因、正向原因、确认原因、风险反证”，并长期统计这些原因的出现频率、组合关系、反证条件和事前可观察性。

## 2. 五号不是股票评分表

五号不输出股票综合分、买入分、推荐分、交易优先级分。五号输出：

```text
背景过程命中
正向原因命中
确认原因命中
风险反证命中
原因口径共振统计
原因组合统计
未识别说明
```

“维度”必须是能解释暴涨前因果链的原因口径，不是涨幅字段、普通突破字段、调试字段、同源重复字段。

## 3. 命名硬规则：不能乱用截图颜色

截图颜色只是用户讲课时的临时标注，绝不能写成正式模型名、字段名、报告名。

正式名称只能用：

```text
核心线
逻辑分析线
```

代码字段只能用：

```text
core_line
logic_analysis_line
```

禁止在正式报告、代码变量、文档标题里把颜色名当定义使用。

## 4. 民间战法与华尔街体系融合原则

五号必须融合两套体系：

```text
用户的民间实战战法
+
华尔街专业市场结构框架
=
统一的底层市场行为原因口径
```

不能只听民间战法，也不能只堆华尔街术语。二者必须映射到底层市场行为。如果民间战法和华尔街表达描述的是同一个市场行为，只能归并成一个原因口径，禁止重复计分、重复论证。

非常重要：

```text
物极必反不是大涨原因。
物极必反只是背景过程，说明前一次极强量价事件失败。
它本身不是好事，不能放进正向原因统计。
只有后面出现修复链条、量能改善、规律性缩量、核心线突破、回踩确认、平量反包等，才可能形成真正的大涨归因。
```

## 5. 周期映射硬规则

用户人工复盘中说的“月线”，在五号聚合K体系中优先映射为：

```text
20日聚合K ≈ 月线级别
60日聚合K ≈ 季线级别
250日聚合K ≈ 年线级别
```

所有“月线倍量、月线平量、月线实体质量、月线双上影共振、月线量价合一”等描述，在五号代码里优先按 20日聚合K 计算。若未来额外使用自然月K，必须单独标注，不能和20日聚合K混算。

## 6. 第一批量化原因口径池

本批口径来自用户对 688260 的人工复盘。688260 不是案例故事，而是用来拆解可复用的暴涨原因维度池。今天先落第一批，后续只做增量扩充和阈值校准。

### BKG001 高量强阳失败背景

角色：背景过程，不算正向原因。

民间表达：物极必反只是过程。最大量大阳线被后续缩量阴线跌坏，说明前一次最强攻击失败。

华尔街映射：failed high-volume impulse / failed acceptance。

量化公式：

```text
1. 在20日聚合K或60日聚合K中，出现局部最高量或局部显眼高量强阳。
2. 强阳要求：局部量比≥1.8，或局部量能分位≥90%。
3. 后续若干聚合K收盘跌破该强阳实体实底或最低点。
4. 若跌破过程伴随缩量，记为“失败后消化背景”；若放量长阴砸穿，改归风险破坏。
```

### BKG002 失败后的二次努力供应线

角色：背景过程，不算正向原因。

民间表达：第一次最大努力失败后，后面再次放出阶段显眼量努力上攻，但未完全解放左侧供应，该高点有研究价值。

华尔街映射：renewed effort with limited result。

量化公式：

```text
1. 先有BKG001。
2. 失败后缩量消化段中，出现局部显眼量K。
3. 该K高点低于或未有效突破前一次高量强阳高点/筹码区。
4. 该高点作为核心线候选的一端，不能直接当正向原因。
```

### CORE001 隔断式带量上影共振核心线

角色：结构原因。

民间表达：两个长期相隔的带量上影高点在同一区域共振，中间可以有阶段性集中穿越，但不能多年乱穿。

华尔街映射：market memory / supply reaction confluence。

量化公式：

```text
1. 右端通常来自BKG002的二次努力高点。
2. 左端寻找远期带量上影高点。
3. 两端价格差≤6%。
4. 两端量能都必须在当时局部环境显眼：局部量比≥1.3，或局部量能分位≥80%。
5. 中间穿越必须集中：穿越跨度≤4根聚合K，或穿越跨度/两端距离≤25%。
6. 中间穿越次数≤6次。
7. 满足后才叫核心线；不能硬画。
```

### VOL001 局部相对量能确认

角色：基础规则。

民间表达：历史量能不能跨年份绝对比较，要看当时是否显眼。

华尔街映射：relative volume / contextual volume significance。

量化公式：

```text
rel_vol = 当前聚合K量 / 前N根聚合K均量
rank_pct = 当前量在局部窗口中的分位
若聚合K历史不足，则下钻周线/日线验证。
```

硬规则：不能因为左侧历史量绝对小于当前量，就否定左侧带量共振级别。

### HVF001 严格相邻并肩高倍量平量

角色：结构原因。

民间表达：不是普通平量，而是大周期严格相邻的高倍量平量。

华尔街映射：volume consistency。

量化公式：

```text
1. 两根K线必须严格相邻，中间不能有任何穿插；不相邻不叫并肩。
2. 两根均为局部显眼高量：至少一根rel_vol≥1.8，或量能分位≥85%。
3. 两根量差：
   强平量：diff≤5%
   合格平量：5%<diff≤8%
   宽松平量：8%<diff≤15%
   diff>15% 不叫平量。
```

### HVF002 第一根高倍量强阳量价合一

角色：结构原因。

民间表达：大量必须配大实体、短上影、强收盘，否则就是有量无价。

华尔街映射：effort vs result。

量化公式：

```text
若20日聚合K vol_ratio 2-3倍：实体涨幅≥15%，body_ratio≥0.50，close_pos≥0.65，上影≤0.35。
若20日聚合K vol_ratio 3-5倍：实体涨幅≥22%，body_ratio≥0.55，close_pos≥0.70，上影≤0.30。
若20日聚合K vol_ratio≥5倍：实体涨幅≥30%，body_ratio≥0.60，close_pos≥0.75，上影≤0.25。
```

反证：高倍量小实体、高倍量长上影、高倍量后快速跌破实底，都不能算正向结构。

### HVF003 第二根不破坏第一根上三分之一

角色：结构原因。

民间表达：第二根可以不如第一根强，但不能破坏第一根强阳。

华尔街映射：high-level acceptance。

量化公式：

```text
第一根强阳实体上三分之一位 = 实体底 + 实体高度 × 2/3。
第二根收盘 ≥ 第一根实体上三分之一位：高质量不破坏。
第二根收盘在实体中位到上三分之一之间：降级。
第二根收盘跌破实体中位：明显降级。
第二根跌破第一根实底：强阳失败。
```

### LOG001 逻辑分析线成立

角色：分析锚点，不等于核心线。

民间表达：这条线解释资金从冷转热、右侧量价演化和大涨前因果链。

华尔街映射：change-of-character anchor。

量化公式：

```text
1. 来源优先为高倍量平量强阳结构的第一根强阳实体实底。
2. 若强阳下影很长，不能机械取最低点，优先取实底。
3. 若只是实体大但无高倍量平量、无右侧共振，只能作为候选，不能硬定。
4. 逻辑分析线不能替代核心线。
```

### REG001 启动后右侧均量抬升

角色：正向原因。

民间表达：启动前量能很差，启动后右侧量能逐渐好转。

华尔街映射：volume regime shift。

量化公式：

```text
left_mean = 启动前6根20日聚合K均量
right_mean = 启动后6根20日聚合K均量
成立条件：right_mean ≥ left_mean × 1.15，且右侧价格中枢不明显下移。
```

### REG002 均量抬升后的量能稳定

角色：正向原因。

民间表达：越来越稳定不是低量死水，而是均量抬升后的稳定。

华尔街映射：lower dispersion after higher participation。

量化公式：

```text
volume_cv = std(volume) / mean(volume)
右侧均量 ≥ 左侧均量 × 1.15
右侧volume_cv ≤ 左侧volume_cv × 0.80
右侧极端量柱数量 ≤ 左侧极端量柱数量
右侧价格中枢 ≥ 左侧价格中枢 × 0.98
```

极端量柱定义：单根量 > 窗口均量×1.8，或 < 窗口均量×0.5。

### PULL001 逻辑分析线跌破后的规律性缩量小阴

角色：正向原因。

民间表达：跌破不一定坏，关键看是否小阴、规律缩量、均量抬升。

华尔街映射：controlled pullback / orderly volume decay。

量化公式：

```text
1. 3根聚合K内至少2根为小阴。
2. 小阴定义：阴线，实体跌幅≤12%，body_ratio≤0.65。
3. 量能满足 V1 > V2 > V3。
4. d1=(V1-V2)/V1，d2=(V2-V3)/V2。
5. d1、d2均在5%-25%。
6. abs(d1-d2)≤8%。
```

### PULL002 缩量回落但均量仍高于左侧

角色：正向原因。

民间表达：下跌量能虽然是阴量，但比左侧均量明显抬升，说明不是死水。

量化公式：

```text
三根小阴均量 ≥ 启动前低迷期均量 × 1.15。
```

### REACT001 逻辑分析线右侧多点共振

角色：正向原因。

民间表达：逻辑分析线画出后，右侧锤子线、实体、上影线、阶段高量K都继续围绕它反应。

华尔街映射：reaction clustering。

量化公式：

```text
1. 逻辑分析线生成后，右侧实体/影线/收盘价接近该线≤5%的反应次数≥3。
2. 至少部分反应点为局部显眼量：rel_vol≥1.2，或量能分位≥75%。
3. 右侧反应权重大于远期普通左侧点，因为右侧量能状态已抬升。
```

### LAD001 规则阳梯量推进

角色：正向原因。

民间表达：阳量逐级递增，递增比例有规律，不是乱放量。

华尔街映射：progressive demand expansion。

量化公式：

```text
观察窗口：连续3-5根同周期聚合K。
1. 成交量逐级递增：V1<V2<V3，或4/5根窗口中连续递增。
2. 单次递增幅度 = Vn/V(n-1)-1。
3. 单次递增幅度在8%-45%。
4. 递增幅度CV≤0.60；CV≤0.35为高度规则。
5. 阳线占优：3根至少2根阳线；4根至少3根阳线。
```

### LAD002 梯量最后一根量价最强

角色：正向原因。

民间表达：最后一根量最大，实体也最大，才是量价合一的最后一击。

量化公式：

```text
1. 最后一根成交量为窗口最大。
2. 最后一根实体涨幅为窗口最大或最强。
3. close_pos≥0.70。
4. upper_shadow_ratio≤0.30。
```

反证：量递增但实体越来越小、上影越来越长，视为高量低效或供应压制。

### CORE002 核心线有效突破

角色：确认原因。

民间表达：核心线不是摸一下，要高级别K线有效打穿。

华尔街映射：valid breakout。

量化公式：

```text
1. 收盘价 ≥ 核心线 × 1.02。
2. 实体在线上方比例 ≥ 50%。
3. close_pos ≥ 0.70。
4. upper_shadow_ratio ≤ 0.30。
5. 成交量 ≥ 突破前6根20日聚合K均量 × 1.15。
```

### CORE003 突破后量能接受

角色：确认原因。

民间表达：突破后量能仍在均量上方，说明不是一日游。

华尔街映射：acceptance above breakout。

量化公式：

```text
核心线突破后3根20日聚合K内，至少2根成交量 ≥ 突破前6根20日聚合K均量。
更强：至少2根成交量 ≥ 前6均量×1.15。
```

### CORE004 突破后未快速跌回核心线

角色：确认原因。

民间表达：突破后不能马上跌回去。

华尔街映射：no fast failed breakout。

量化公式：

```text
20日聚合K：突破后2根内，不能出现收盘价 < 核心线 × 0.97。
日线：突破后5个交易日内，不能连续2天收盘在核心线下方，或收盘跌破核心线2%以上。
```

### CORE005 核心线回踩支撑确认

角色：确认原因。

民间表达：核心线由压力转支撑。

华尔街映射：breakout-retest。

量化公式：

```text
1. 前面已有效突破核心线。
2. 后续回踩时：low ≤ 核心线×1.08 且 high ≥ 核心线×0.94。
3. 收盘不有效跌破：close ≥ 核心线×0.97。
4. 最好后续1-2根出现修复K。
```

### REV001 核心线支撑后的平量阳包阴

角色：确认原因。

民间表达：核心线支撑后出现平量阳包阴，说明稳定资金修复反包。

华尔街映射：stable participation reversal。

量化公式：

```text
1. 前一根为阴线，后一根为阳线。
2. 阳线收盘 ≥ 前阴实体顶。
3. 阳线量与前阴量差≤15%。
   ≤5%：强平量阳包阴；≤8%：合格；≤15%：宽松。
4. 附近5根量能CV≤0.40，CV≤0.25为高度稳定。
5. 该结构发生在核心线支撑之后，意义更高。
```
